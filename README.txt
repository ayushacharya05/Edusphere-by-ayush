== Description ==
EduSphere by Ayush is a modern, professional WordPress theme purpose-built
for schools, colleges and universities. It comes with dedicated modules for
everything a modern educational institution's website needs, without
requiring any page builder or premium plugin.

== Features ==
* Admissions-focused homepage: hero, stats bar, programs, notices, events,
  testimonials, faculty highlights and latest news.
* Notice Board (custom post type) with "Important" flag, categories, and
  optional PDF/attachment links. Important notices auto-scroll in the
  header ticker.
* Academic Calendar / Events (custom post type) with date, time and
  location fields, sorted chronologically.
* Faculty & Staff Directory (custom post type) with designation,
  qualification, email, LinkedIn and Department taxonomy.
* Programs & Courses (custom post type) with level, duration and seats
  meta fields, organized by Department taxonomy.
* Testimonials (custom post type) with role and star rating.
* Built-in Blog/News section for school updates and stories.
* Sticky header with top contact bar, social links, search overlay and
  a mobile-friendly slide-in navigation menu.
* Admissions "Apply Now" call-to-action wired to a customizable link.
* Customizer options for phone, email, address, social URLs and the
  admissions link — no code editing required.
* Widgetized sidebar plus four footer widget columns and a newsletter
  signup form.
* Fully responsive, accessible markup, translation-ready, and built with
  clean semantic HTML5 + vanilla JS (no jQuery dependency).
* One-click "EduSphere Setup" page (under Appearance) to install sample
  notices, events, faculty, programs and testimonials for demo purposes.

== Installation ==
1. Upload the edusphere-by-ayush folder to /wp-content/themes/
2. Activate the theme from Appearance > Themes.
3. Go to Appearance > EduSphere Setup to install sample content (optional).
4. Set your contact info and social links under Appearance > Customize >
   School Contact Info.
5. Create menus under Appearance > Menus and assign them to the Primary,
   Top Bar, and Footer menu locations.
6. Add content via the Notices, Events, Faculty, Programs and Testimonials
   menu items in the WordPress admin sidebar.
